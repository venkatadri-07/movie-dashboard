# 🎬 Movie Analytics Dashboard | Power BI

## 📖 About the Project
The **Movie Analytics Dashboard** is a data visualization project built using **Microsoft Power BI**.  
This dashboard provides insights into movie performance by analyzing key metrics such as **revenue, ratings, popularity, genres, and languages**.

The aim of this project is to convert raw movie data into **interactive visual insights** that help understand trends in the movie industry.

---

## 📊 Dashboard Highlights
✔ **Total Movies Analysis** – Overview of total movies available in the dataset  
✔ **Revenue Analysis** – Revenue generated by different movies and genres  
✔ **Rating vs Popularity** – Relationship between audience ratings and popularity  
✔ **Movies by Genre** – Distribution of movies across different genres  
✔ **Revenue by Genre** – Identifying which genres generate the highest income  
✔ **Language Distribution** – Movies categorized by language  
✔ **Year-wise Movie Releases** – Trend analysis of movie releases over time  

---

## 📈 Key Insights
- Drama and Comedy are the most common movie genres.
- Action movies generate significant revenue compared to other genres.
- Movie releases have increased significantly in the past two decades.
- English is the most common language in the dataset.
